# Basmo Airways - Airline Travel Reservation System

A JavaFX desktop application for booking and managing 
airline travel reservations with a clean professional UI.

## Features
- Book one-way and round-trip flights
- Auto-generate unique booking reference codes
- Search existing bookings by reference code
- Full input validation with error highlighting
- Seat class selection (Economy, Business, First Class)
- Meal preference and baggage options
- Special assistance request
- Displays a formatted travel ticket after booking
- Supports African nationalities

## Technologies Used
- Java
- JavaFX (Scene, Stage, GridPane, BorderPane)
- JavaFX UI Controls (ComboBox, DatePicker, RadioButton)
- JavaFX CSS Styling
- Java Collections (HashMap)
- Java SecureRandom (Reference Code Generation)
- NetBeans IDE

## How to Run
- Make sure you have Java JDK 8 or higher installed
- Make sure JavaFX is configured in your IDE
- Open the project in NetBeans
- Run the AirtimeTravelReservation.java file

Note: This project requires JavaFX libraries to be added 
to your project. JavaFX is not included by default in 
Java 11 and above.
